# Design Agency Homepage

This project is a responsive Design Agency Homepage developed using React.js, Next.js 13, TypeScript, Bootstrap, and custom CSS.

The project was created as part of a React.js Developer Internship assignment.

## Features

* Responsive Hero Section
* Services Section with service cards
* Portfolio Section with project showcase
* Contact Form with basic validation
* Success message popup
* Hover effects and animations

## Technologies Used

* React.js
* Next.js 13+
* TypeScript
* Bootstrap 5
* CSS

## Setup Instructions

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```bash
http://localhost:3000
```
